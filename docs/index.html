SGuard

Lightweight guard/validation helpers for .NET
GitHub NuGet Discussions
Why SGuard?

SGuard provides concise, readable guard clauses for parameter and state validation in .NET apps. It favors developer ergonomics and performance with minimal dependencies and a clean API surface.

  • Simple API: Throw.IfNull(obj), Throw.IfNullOrEmpty(str)
  • Fluent chaining and callbacks
  • Minimal overhead, friendly to hot paths
  • Targets modern .NET (and can multi-target for compatibility)
Install
dotnet add package SGuard
Quick Start
using SGuard;

public static void Process(string name, int age)
{
    Throw.IfNullOrEmpty(name, "Name is required");
    Throw.IfLessThan(age, 18, "Age must be 18+");

    // ...
}
